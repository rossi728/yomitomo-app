おうちのかたへ

音声対話機能について大切なお知らせ

🗣️ 音声対話機能とは

お子さまが絵本のキャラクターと、マイクを使って会話できる機能です。

📡 送信される情報

  • お子さまがマイクに話した音声データ
  • 選択された絵本のキャラクター名(例:ももたろう)

🚫 送信されない情報

  • お名前・メールアドレス・電話番号
  • 位置情報・端末情報
  • Apple ID の情報

🌐 送信先

音声データは、弊社サーバー(api.yomitomo.com)を経由して、OpenAI 社(アメリカ合衆国)の Realtime API に送信されます。

OpenAI 社は、本アプリから送信された音声データを AI モデルの学習には使用しません。データは不正利用監視のため最大 30 日間保持された後、削除されます。

🔒 データの保存について

  • 弊社では、音声や対話内容を保存しません
  • 通信はすべて暗号化されています

⚙️ 同意の取り消し

一度同意しても、アプリ内の「設定」から、いつでも取り消せます。取り消すと音声対話機能は停止します(絵本の閲覧は引き続き可能です)。
